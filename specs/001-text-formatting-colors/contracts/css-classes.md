# Contract: CSS Classes for mIRC Formatting

**File**: `apps/retro_hex_chat_web/assets/css/dark-theme.css`
**Purpose**: Define CSS classes for rendering mIRC format codes in chat messages.

## Formatting Classes

Applied to `<span>` elements generated by `Chat.Formatter.to_safe_html/1`.

### Text Style Classes

```css
.irc-bold        { font-weight: bold; }
.irc-italic      { font-style: italic; }
.irc-underline   { text-decoration: underline; }
.irc-strikethrough { text-decoration: line-through; }
.irc-reverse     { /* JS/server sets explicit fg/bg swap based on context */ }
```

Note: `.irc-underline` and `.irc-strikethrough` can be combined:
```css
.irc-underline.irc-strikethrough { text-decoration: underline line-through; }
```

### Reverse Video

For `.irc-reverse`, the formatter generates explicit `irc-fg-N` and `irc-bg-N` classes with swapped values based on the current state. If no colors are set, the default chat text color and background are swapped:

```css
.irc-reverse-default {
  color: var(--surface-bg);
  background-color: var(--text-color);
}
```
