/query

Open a private message tab with a user without sending any message. Unlike /msg, this just creates the conversation window.

Syntax

/query <nickname>

Parameters

nickname — The user to open a PM tab with.

How It Works

Creates or switches to a PM conversation tab with the specified user. No message is sent — it simply opens the window so you can start typing. If you already have a PM tab open with that user, it switches to it. If you want to open the tab and send a message in one step, use /msg instead.

Examples

/query Alice
/query Bob
