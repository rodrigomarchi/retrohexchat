/setmotd

Set the server's Message of the Day — a welcome message shown to all users when they connect.

Syntax

/setmotd <text>

Parameters

text — The MOTD content to display.

How It Works

Sets or replaces the server MOTD. After setting it, all newly connecting users will see this message in the Status tab when they join. Existing connected users are not notified — only new connections see the MOTD. Any previous MOTD is replaced.

Rules & Limits

  • Server administrator only
  • Message text is required

Examples

/setmotd Welcome to RetroHexChat! Please read the rules in #rules.
