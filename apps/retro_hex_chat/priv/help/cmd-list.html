/list

Browse all public channels on the server in a sortable, filterable dialog window.

Syntax

/list

How It Works

Opens the Channel List dialog showing all available channels with their user counts and topics. You can sort columns by clicking the headers and filter channels by name. Double-click a channel to join it. Secret (+s) channels are completely hidden from the list. Private (+p) channels show their name but not their topic.

Examples

/list
