Message Edit

Edit your own messages within 5 minutes of sending to fix typos or correct content.

How to Edit

↑ key: Press the up arrow in an empty input to edit your last message. The input fills with the message text, and the message gets a dashed border indicating edit mode.

Submitting & Canceling

Enter: Submit the edit. The message updates for all viewers with an "(editado)" tag.
Escape: Cancel the edit and restore the input to its previous state.
Empty edit: Submitting with empty text opens a delete confirmation dialog.

Edited Indicator

Edited messages show a small "(editado)" tag. Hover over it to see the edit timestamp.

Limitations

You can only edit your own messages. The 5-minute time limit is enforced server-side. Editing preserves any reply context (if the message was a reply).
