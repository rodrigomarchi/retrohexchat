/ban

Permanently block a user from the current channel. The banned user is immediately removed and cannot rejoin until the ban is lifted.

Syntax

/ban <nickname>

Parameters

nickname — The user to ban from the channel.

How It Works

The user is kicked from the channel and a ban is set, preventing them from rejoining. The ban stays in effect until a channel operator removes it using /mode -b nickname. Other channel members see that the user was banned.

Rules & Limits

  • Requires channel operator status
  • Cannot ban yourself
  • To remove a ban, use /mode -b nickname

Examples

/ban Spammer
