/motd

View the server's Message of the Day — a welcome message from the server administrators that is also shown automatically when you first connect.

Syntax

/motd

How It Works

Displays the current server MOTD in the Status tab. The MOTD is shown automatically when you connect, but you can use this command to view it again at any time. If no MOTD has been set by the administrators, you'll see a "No MOTD has been set" message.

Examples

/motd

See Also

/setmotd · /clearmotd · Special Messages
