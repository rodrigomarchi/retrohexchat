/kick

Remove a user from the channel. Unlike /ban, the kicked user can rejoin immediately unless a ban is also set.

Syntax

/kick <nickname> [reason]

Parameters

nickname — The user to kick.
reason — Optional message explaining why (visible to the channel).

How It Works

The user is immediately removed from the channel. All channel members see the kick along with the optional reason message. The kicked user can rejoin the channel immediately unless they are also banned. If you want to permanently prevent someone from returning, use /ban instead.

Rules & Limits

  • Requires half-operator (%) status or higher
  • Cannot kick users with equal or higher rank than you
  • An operator cannot kick another operator — only the owner can
  • The target must be in the channel

Examples

/kick Spammer
/kick Spammer Flooding the channel
/kick Troll Please follow the rules
