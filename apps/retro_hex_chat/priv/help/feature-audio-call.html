Audio Call

Start peer-to-peer audio calls with another user in a P2P session.

Starting a Call

In the P2P session, click Audio Call. The other user must accept the request.

Mute/Unmute Microphone

Click the Mute button to mute your microphone. The other user will see an indication that you are muted.

End Call

Click End Call to finish. Both users return to the lobby.

Indicators

The timer shows the call duration. The quality indicator shows 4 levels: Excellent, Good, Fair, Poor.
