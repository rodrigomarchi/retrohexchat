/clear

Clear all messages from the current chat window, giving you a fresh screen. This cannot be undone.

Syntax

/clear

How It Works

Removes all visible messages from whichever window you're currently viewing — a channel, a PM conversation, or the Status tab. This only affects your screen; other users' windows are not changed. The messages are gone permanently and cannot be recovered by scrolling up.

Examples

/clear

See Also

/quit
