/unban

Remove a ban from a user, allowing them to rejoin the channel.

Syntax

/unban <nickname>

Parameters

nickname — The user to unban.

How It Works

Removes the ban entry for the specified user. Once unbanned, the user can rejoin the channel normally. For registered channels, the unban is persisted. You can also unban using /mode -b nickname.

Rules & Limits

  • Requires channel operator or owner status

Examples

/unban user123
