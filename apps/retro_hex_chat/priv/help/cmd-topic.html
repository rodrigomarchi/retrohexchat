/topic

View or change the channel topic — the text displayed in the topic bar at the top of the channel window.

Syntax

/topic [new topic text]

Parameters

new topic text — The new topic to set. If omitted, shows the current topic.

How It Works

Running /topic with no arguments displays the current channel topic and who set it. Providing text sets a new topic that appears in the topic bar at the top of the channel for all members to see. The topic bar is always visible, making it a good place for channel rules, links, or the current discussion subject.

Rules & Limits

  • If the channel has +t mode (topic lock), only operators and owners can change the topic
  • Without +t, any channel member can change the topic
  • Must be in a channel to use this command

Examples

/topic
/topic Welcome to #elixir! Today: LiveView discussion
/topic Rules: Be respectful, no spam
