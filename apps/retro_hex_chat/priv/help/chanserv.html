ChanServ Overview

ChanServ is the channel registration service. It lets you register and manage channels with persistent access control.

Common Commands

/cs register #channel             — Register a channel (must be op)
/cs access #channel add <nick> <level> — Add user to access list
/cs access #channel remove <nick>      — Remove from access list
/cs access #channel list               — View access list
/cs drop #channel                      — Unregister a channel

Access Levels

op — Full operator rights. voice — Can speak in moderated (+m) channels.
