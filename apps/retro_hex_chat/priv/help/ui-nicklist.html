User List

The user list appears in the conversations sidebar (left pane) under the active channel, showing all users sorted by role.

Roles

Users are sorted by role priority and distinguished by icon and color:
Owners (~): Purple, bold. Channel founders.
Operators (@): Red, bold. Can manage the channel.
Half-Ops (%): Green, bold. Limited moderation.
Voiced (+): Blue. Can speak in moderated channels.
Regular: Default color.

User Count

The active channel shows a user count in parentheses next to the channel name, e.g. #lobby (12).

Context Menu

Click a nickname to access Query, Whois, Kick, Ban, Mode, Add to Contacts, and Set Nick Color options. Double-click to open a private message.
