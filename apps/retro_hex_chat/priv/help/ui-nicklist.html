Nicklist

The nicklist (right pane) shows all users in the current channel, grouped by role.

Header

The nicklist header shows "Users (N)" with the total count and a collapse button (◀) to hide the panel. You can also toggle it via View > Toggle Nicklist.

Groups

Only groups with members are displayed:
Owners (~): Purple, bold. Channel founders.
Operators (@): Red, bold. Can manage the channel.
Half-Ops (%): Green, bold. Limited moderation.
Voiced (+): Blue. Can speak in moderated channels.
Regular: Default color.

Context Menu

Right-click a nickname to access Query, Whois, Kick, Ban, Mode, Add to Contacts, and Set Nick Color options.
