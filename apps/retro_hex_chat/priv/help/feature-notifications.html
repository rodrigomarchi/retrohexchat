Notifications

The notification system provides coordinated alerts when activity occurs in channels or PMs you are not currently viewing.

Notification Channels

When a notification fires, multiple output channels activate simultaneously:

  • Toast popup — Bottom-right toast with sender and preview (max 3 visible, auto-dismiss 5s)
  • Sound — Event-appropriate notification sound
  • Title flash — Browser title alternates when tab is in background
  • Browser notification — Native OS notification (if permission granted)
  • Favicon badge — Red dot on browser tab icon
  • Conversations badge — Unread count and highlight indicator

Trigger Rules

Configure what triggers notifications in Options > Notifications: mentions, PMs, all channel messages, or joins/leaves.

Per-Channel Levels

Each channel can be set to Normal (all notifications), Mentions only (only when your nick is mentioned), or Mute (no notifications). PMs always notify.

Active Channel Suppression

If you are viewing a channel, notifications for that channel are suppressed — you can already see the messages.
