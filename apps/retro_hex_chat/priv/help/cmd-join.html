/join

Join a channel. Creates the channel if it does not exist.

Syntax

/join <#channel> [key]

Parameters

#channel — Channel name (must start with #).
key — Optional channel key if the channel has mode +k set.

Examples

/join #elixir
/join #secret mykey

Auto-Join

For registered users, joining a channel automatically adds it to your auto-join list. The channel will be rejoined on your next connection. See Auto-Join Channels for details.
