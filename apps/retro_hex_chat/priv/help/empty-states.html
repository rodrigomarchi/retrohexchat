Empty States

When a UI container has no content, a friendly placeholder message appears with guidance:

Channel Messages

An empty channel shows a welcome message and tips for getting started.

Nicklist

Shows "Nobody here" when you are the only user (or the channel is truly empty).

Conversations

When no channels are joined, shows a hint to use /join and an "Browse channels" button.

URL Catcher

Shows "No URLs captured" until URLs appear in chat.

Notes

Empty state messages disappear automatically when content arrives. The text is not selectable.
