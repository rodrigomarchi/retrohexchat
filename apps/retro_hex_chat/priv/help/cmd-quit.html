/quit

Disconnect from the server and return to the connect screen. Other users see your optional quit message in all channels you were in.

Syntax

/quit [message]

Parameters

message — Optional goodbye message shown to other users.

How It Works

You are disconnected from the server and returned to the connect screen. All your channel memberships end, and other users see a quit notification with your optional message. Your session is fully ended — to rejoin, you'll need to connect again from the connect screen.

Examples

/quit
/quit Goodnight everyone!
