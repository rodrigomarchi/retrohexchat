Message Delete

Delete your own messages within 5 minutes of sending.

How to Delete

Right-click your message and select "Apagar mensagem". A confirmation dialog appears with Confirmar/Cancelar buttons. Press Escape or click Cancelar to dismiss.

What Happens

The message is replaced with "[mensagem removida]" in italic text for all viewers. The original content is preserved in the database (soft-delete) for audit purposes.

Limitations

You can only delete your own messages. The 5-minute time limit is enforced server-side. Channel operators cannot delete other users' messages.

Replies to Deleted Messages

If someone replied to a message that is later deleted, the reply quote shows "[mensagem removida]" instead of the original content.
