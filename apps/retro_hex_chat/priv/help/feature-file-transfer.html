File Transfer

Send files directly to another user via a peer-to-peer connection (WebRTC DataChannel).

Sending a File

/sendfile <nickname>

Or in the P2P session, click Send File and select the file.

Receiving

When you receive an offer, you see the file name and size. Click Accept or Reject.

Progress

The progress bar shows: percentage, speed, and estimated time. You can cancel at any time.

Integrity Check

After the transfer, the SHA-256 hash is verified automatically. If it fails, you can try again.

Pause and Resume

If the connection drops briefly, the transfer pauses automatically and resumes when the connection is re-established.

Restrictions

Blocked extensions: .exe, .bat, .cmd, .com, .msi, .scr, .pif, .vbs, .js, among others. Maximum size is configurable by the server (default: 500 MB).
