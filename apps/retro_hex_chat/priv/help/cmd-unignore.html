/unignore

Stop ignoring a user you previously blocked with /ignore, making their messages visible to you again.

Syntax

/unignore <nickname>

Parameters

nickname — The user to stop ignoring.

How It Works

Removes the user from your ignore list. Their messages, PMs, actions, and invites (whichever were being blocked) will be visible again immediately. To see who you're currently ignoring, run /ignore with no arguments first.

Examples

/unignore SpamBot
/unignore Alice
