/clearmotd

Remove the server's Message of the Day so that new users connecting won't see any MOTD.

Syntax

/clearmotd

How It Works

Clears the server MOTD that is normally displayed to users when they connect. After clearing, the MOTD section is simply skipped during connection. You can set a new MOTD at any time with /setmotd.

Rules & Limits

  • Server administrator only

Examples

/clearmotd
