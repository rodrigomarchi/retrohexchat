/setwelcome

Set a message shown once to each user when they join the current channel. Great for posting rules or helpful information for newcomers.

Syntax

/setwelcome <message>

Parameters

message — The welcome text to display to joining users.

How It Works

After setting a welcome message, any user who joins the channel sees it once per session. If they leave and rejoin, they won't see it again during the same session. The person who set the welcome message does not see it themselves. Each channel has its own independent welcome message.

Rules & Limits

  • Channel operator or owner only
  • Must be used inside a channel
  • Message text is required

Examples

/setwelcome Welcome to #elixir! Please be respectful and use English.
