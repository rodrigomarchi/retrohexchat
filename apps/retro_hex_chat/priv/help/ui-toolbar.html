Toolbar

The toolbar provides quick access to all RetroHexChat features through icon buttons, organized in six groups separated by dividers.

Button Groups

Connection

  • Connect / Disconnect — Connect to the server or disconnect from it.

View

  • Channel List — Browse and join available channels.
  • Toggle Treebar — Show or hide the left-side channel tree.
  • Toggle Nicklist — Show or hide the right-side user list.
  • Find — Open the search bar to find text in the current channel.

Tools

  • Address Book — Manage contacts, notify list, and nick colors.
  • Highlight Words — Configure words that trigger message highlighting.
  • Ignore List — Manage ignored users.
  • URL Catcher — View URLs shared across channels.
  • Channel Central — View and manage channel settings, bans, and modes.
  • Log Viewer — Browse chat history and export logs.
  • Perform — Configure auto-perform commands and auto-join channels.
  • Sounds — Configure sound notifications for events.
  • CTCP Settings — Configure Client-to-Client Protocol responses.
  • Flood Protection — Configure anti-flood and auto-ignore settings.
  • Alias Editor — Create custom command aliases.
  • Custom Menus — Add custom items to context menus.
  • Auto-Respond — Configure automatic response rules.
  • Settings — Open the options dialog for display, font, and behavior preferences.

Notifications

  • DND (Do Not Disturb) — Toggle Do Not Disturb mode. When active, notifications are suppressed.
  • Bell — Open the Notification Center. A badge shows the count of unread notifications.

Help

  • Help Topics — Opens the help page in a new tab.
