/help

View help for any command, or browse all available commands. Your gateway to learning how RetroHexChat works.

Syntax

/help [command]

Parameters

command — Optional command name (without the / prefix). If omitted, shows a list of all commands.

How It Works

Running /help with no arguments displays a categorized list of all available commands with brief descriptions. Adding a command name (like /help join) shows detailed help for that specific command. You can also press F1 at any time to open the Help Topics window, or use the Help menu in the menu bar to search all help topics.

Examples

/help
/help join
/help mode
/help ns
