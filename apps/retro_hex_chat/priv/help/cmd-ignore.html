/ignore

Add a user to your ignore list, hiding their messages from your view. The ignored user receives no notification.

Syntax

/ignore                        — Show your ignore list
/ignore <nick>                 — Ignore all content from nick
/ignore <nick> <type>           — Ignore specific type only
/ignore <nick> <type> <duration> — Timed ignore

Types

all      — All messages, PMs, actions (default)
messages — Channel messages only
pms      — Private messages only
actions  — /me actions only
invites  — Channel invites only

Duration Format

5m  — 5 minutes
2h  — 2 hours
1d  — 1 day

Examples

/ignore SpamBot
/ignore AnnoyingGuy pms
/ignore LoudPerson all 5m
