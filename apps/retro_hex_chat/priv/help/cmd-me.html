/me

Send an action message that describes what you're doing. It appears to everyone in the channel as * YourNick does something.

Syntax

/me <action text>

Parameters

action text — What you're doing (at least one word required).

How It Works

Instead of a regular chat message like <YourNick> hello, an action message appears as * YourNick waves hello with special formatting. Action messages support the same text formatting as regular messages (bold, italic, colors, etc.).

Examples

/me waves hello
/me is thinking...
/me grabs some coffee
