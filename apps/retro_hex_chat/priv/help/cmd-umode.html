/umode

Toggle personal user modes that affect what messages you receive. User modes are settings that apply to you, not to a channel.

Syntax

/umode <+/-mode>

Parameters

+/-mode — Use + to enable or - to disable a user mode.

Available Modes

+w — Receive wallops messages (broadcast messages from server operators). These appear in the Status tab.
-w — Stop receiving wallops messages.

How It Works

User modes are personal settings that only affect you. Currently, the only available user mode is +w (wallops), which opts you in to receiving operator broadcast messages sent via /wallops. By default, wallops are disabled.

Rules & Limits

  • User modes are session-scoped — they reset when you disconnect

Examples

/umode +w
/umode -w
