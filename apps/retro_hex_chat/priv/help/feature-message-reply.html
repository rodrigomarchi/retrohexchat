Message Reply

Reply to any message to create a visual quote linking your response to the original.

How to Reply

Context menu: Right-click a message and select "Responder".
Hover button: Hover over a message and click the ↩ button that appears.

Compose Bar

A bar appears above the input showing the original author and a preview of their message. Click ✕ to cancel the reply.

Visual Format

Replies appear with a quoted block showing the original message, with your response below. Click the quoted block to scroll to and highlight the original message.

Edge Cases

If the original message was deleted, the quote shows "[mensagem removida]". Long messages are truncated in the reply preview. Editing the original message updates the quote in all replies.
