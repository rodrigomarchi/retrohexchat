/p2p

Inicia uma sessao P2P (peer-to-peer) com outro usuario registrado.

Syntax

/p2p <nickname>

Parameters

nickname — Nick do usuario com quem iniciar a sessao.

Detalhes

Voce deve estar identificado (registrado). O usuario alvo tambem precisa ser registrado. Nao e possivel ter mais de uma sessao ativa com o mesmo usuario.

Examples

/p2p Mario
