/p2p

Start a direct peer-to-peer session with another user. P2P sessions let you transfer files, make audio calls, or start video calls — all directly between your browsers.

Syntax

/p2p <nickname>

Parameters

nickname — The user to start a P2P session with.

How It Works

Running /p2p creates a session and both users enter a lobby window. From the lobby, you can start file transfers, audio calls, or video calls. All data flows directly between your browsers (peer-to-peer) without going through the server, ensuring privacy. If you specifically want to start an audio call or file transfer, you can use /call or /sendfile instead — they create the P2P session and automatically start the respective action.

Rules & Limits

  • Both you and the target must be registered and identified (via /ns identify)
  • Cannot start a P2P session with yourself
  • Only one active session per user pair at a time

Examples

/p2p Mario
/p2p Alice
