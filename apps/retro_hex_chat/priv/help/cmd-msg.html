/msg

Send a private message to another user. This opens a PM conversation tab for both of you.

Syntax

/msg <nickname> <message>

Parameters

nickname — The user to message.
message — The text to send.

How It Works

Your message is delivered privately — only you and the recipient can see it. A PM tab opens on both sides for the conversation. The recipient receives a notification if they have notifications enabled. If you just want to open a PM tab without sending a message, use /query instead.

Rules & Limits

  • The recipient must be online
  • Both nickname and message are required
  • Can be blocked if the recipient has you on their ignore list

Examples

/msg Alice Hey, how are you?
/msg Bob Check out #project
