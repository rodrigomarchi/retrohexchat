Options Dialog

The Options dialog (Ctrl+Shift+O) lets you configure display and notification preferences.

Panels

Display: Toggle visibility of toolbar, treebar, switchbar, and status bar.
Notifications: Global toggles, trigger rules, per-channel levels, DND, privacy mode.

Apply / OK / Cancel

OK applies changes and closes the dialog. Apply applies changes and keeps the dialog open. Cancel discards unsaved changes.

Persistence

Registered users' preferences persist across sessions. Guest preferences last for the current session only.
