/call

Inicia uma sessao P2P com chamada de audio.

Syntax

/call <nickname>

Parameters

nickname — Nick do usuario para chamar.

Detalhes

Funciona como /p2p, mas automaticamente solicita uma chamada de audio quando ambos os usuarios entram no lobby. O peer precisa aceitar a chamada.

Examples

/call Mario
