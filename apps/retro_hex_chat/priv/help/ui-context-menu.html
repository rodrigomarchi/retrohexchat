Context Menu

Right-click a nickname in the nicklist to open the context menu.

Options

Query: Open a PM with the user.
Whois: Look up user information.
Add to Contacts: Save to your address book.
Set Nick Color: Override the display color for this nick.
Ignore / Unignore: Block messages from the user.

P2P Actions (registered users only)

Sessao P2P: Start a generic P2P session.
Chamada de Audio: Start an audio call.
Chamada de Video: Start a video call.
Enviar Arquivo: Send a file.
These items are disabled if the target is not registered or is yourself.

Operator Actions

Kick / Ban: Moderation (operators only).
Give Op (+o) / Give Voice (+v): Mode changes (operators only).
