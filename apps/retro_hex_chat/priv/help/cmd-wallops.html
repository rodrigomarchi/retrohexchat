/wallops

Send a message to all users who have opted in to receive operator announcements by enabling +w user mode.

Syntax

/wallops <message>

Parameters

message — The broadcast message to send.

How It Works

Wallops messages are delivered to all users who have enabled the +w user mode (via /umode +w). The message appears in each recipient's Status tab. Unlike /announce, wallops can be opted out of — users who haven't enabled +w won't see them. This makes wallops suitable for non-critical operator announcements.

Rules & Limits

  • Server operator or administrator only
  • Only users with +w mode enabled will see the message

Examples

/wallops Server maintenance in 10 minutes
/wallops New chat features available — check #announcements
