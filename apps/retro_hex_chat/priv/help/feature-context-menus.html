Context Menus

Right-click on elements in the chat area and treebar to access context-specific actions.

Nick Menu (Chat Area)

Right-click a nickname in a chat message to see: Private Message, Whois, Copy Nick, Ignore/Unignore, Add to Address Book, Set Nick Color. Registered users also see P2P actions: P2P Session, Audio Call, Video Call, Send File (disabled if target is not registered or is yourself). Channel operators also see: Kick, Ban, Give Voice (+v), Give Op (+o).

URL Menu

Right-click a URL in a chat message to see: Open Link, Copy URL, Save to URL List.

Channel Menu

Right-click a #channel reference in a chat message to see: Join Channel (disabled if already joined), Copy Channel Name, Channel Info.

Message Menu

Right-click the general chat area to see: Copy Message, Copy Selected Text, Quote/Reply (disabled), Ignore Sender. If the message contains URLs, Open Link and Copy URL also appear.

Treebar Menu

Right-click a channel in the treebar to see: Mark as Read, Mute/Unmute Channel, Copy Name, Leave Channel, Channel Settings.

Keyboard Navigation

Use Arrow Up/Down to navigate menu items, Enter to select, and Escape to close.

Notes

Menus automatically reposition if they would go off-screen. The browser's default right-click menu is preserved in the input field. Self-targeting actions (Kick, Ban, Ignore, P2P actions on yourself) appear disabled.
