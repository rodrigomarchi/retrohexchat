/perform

Manage your perform list — commands that auto-execute when you connect.

Syntax

/perform [list|add|remove|move|clear] [args]

Subcommands

list              — Show all perform commands
add <command>      — Append a command to the perform list
remove <position>  — Remove command at the given position
move <from> <to>   — Reorder a command from one position to another
clear             — Remove all perform commands

Notes

Commands execute sequentially on connect when the perform list is enabled. Passwords in /ns identify commands are masked in the list display for security.

Examples

/perform list
/perform add /ns identify mypassword
/perform remove 2
/perform move 3 1
/perform clear
