/popups

Open the Custom Menus dialog to add, edit, or remove custom right-click menu items for the nick list and channels.

Syntax

/popups

How It Works

Opens a dialog where you can customize the right-click context menus. You can add menu items that execute any command when clicked — for example, adding a "Whois" item to the nick list that runs /whois $1 when you right-click a user. Custom menu items appear alongside the built-in items in the context menus.

Rules & Limits

  • Registered users: custom menus are saved across sessions
