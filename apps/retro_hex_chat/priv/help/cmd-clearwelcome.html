/clearwelcome

Remove the welcome message from the current channel so that new users joining won't see one.

Syntax

/clearwelcome

How It Works

Clears the welcome message that is shown to users when they join the current channel. After clearing, no welcome message is displayed until a new one is set with /setwelcome.

Rules & Limits

  • You must be a channel operator or owner
  • Must be used inside a channel

Examples

/clearwelcome
