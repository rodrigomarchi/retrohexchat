/announce

Send an urgent broadcast message to every connected user. This is for critical server-wide announcements that everyone must see.

Syntax

/announce <message>

Parameters

message — The announcement text to broadcast to all users.

How It Works

The announcement appears in each user's currently active window with distinctive bold styling. Unlike regular messages, announcements bypass all ignore filters — every connected user will see it regardless of their ignore settings. Use this for important server-wide notices like scheduled maintenance or rule changes.

Rules & Limits

  • Server administrator only — regular users cannot send announcements
  • Cannot be filtered or ignored by recipients

Examples

/announce Server will restart at midnight UTC
/announce Please read the updated rules in #rules
