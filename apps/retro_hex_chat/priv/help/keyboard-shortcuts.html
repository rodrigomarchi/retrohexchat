Keyboard Shortcuts

Navigation

Ctrl+Shift+/    — Shortcut Cheatsheet
Ctrl+Shift+F    — Find / Search
Ctrl+Shift+]    — Next Window
Ctrl+Shift+[    — Previous Window
Ctrl+Shift+1..9 — Switch to Window 1-9
Escape          — Close search bar / dialog

Windows & Dialogs

Ctrl+Shift+A  — Address Book
Ctrl+Shift+H  — Highlight Words
Ctrl+Shift+G  — Ignore List
Ctrl+Shift+L  — Log Viewer
Ctrl+Shift+O  — Options
Ctrl+Shift+E  — Perform Dialog
Ctrl+Shift+S  — URL Catcher
Ctrl+Shift+W  — Help

Clipboard

Ctrl+C        — Copy selected text

Text Formatting

Ctrl+Shift+B  — Bold
Ctrl+Shift+Y  — Italic
Ctrl+Shift+U  — Underline
Ctrl+Shift+D  — Color
Ctrl+Shift+V  — Reverse
Ctrl+Shift+X  — Reset formatting

Input & Autocomplete

Enter         — Send message / select autocomplete item
Shift+Enter   — Insert newline in input
Up            — Edit last message (when input is empty) / command history
Up / Down     — Command history / navigate autocomplete dropdown
Ctrl+Up       — Browse history (saves draft)
Ctrl+Down     — Browse history forward (restores draft)
Ctrl+R        — Reverse search history
Tab           — Select autocomplete item / cycle nick completion
Escape        — Cancel edit mode / dismiss autocomplete / tooltip
/             — Open command autocomplete (at start of input)
@             — Open nick autocomplete (at word boundary)
#             — Open channel autocomplete (at word boundary)

Context Menus

Arrow Down    — Next menu item
Arrow Up      — Previous menu item
Enter         — Select focused menu item
Escape        — Close context menu

Search Navigation

Arrow Down    — Next search result (in search bar)
Arrow Up      — Previous search result (in search bar)

Emoji

Emoji Picker  — Click smiley button in formatting toolbar
