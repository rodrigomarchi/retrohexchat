/notify

Track when specific users come online or go offline. Build a buddy list so you never miss when your friends connect.

Syntax

/notify
/notify add <nickname> [note]
/notify remove <nickname>
/notify edit <nickname> <note>
/notify list

Parameters

nickname — The user to track.
note — Optional text to remind you who this person is.

How It Works

When someone on your notify list connects or disconnects, you receive a notification in the Status tab. You can add personal notes to help remember who each person is. Running /notify with no arguments opens the Notify List dialog where you can manage your list visually.

Rules & Limits

  • Registered users: list is saved across sessions
  • Guests: list lasts only for the current session

Examples

/notify add Alice My friend from #elixir
/notify add Bob
/notify edit Alice Best friend
/notify remove Bob
/notify list
