/sendfile

Inicia uma sessao P2P para transferencia de arquivo.

Syntax

/sendfile <nickname>

Parameters

nickname — Nick do usuario para enviar o arquivo.

Detalhes

Funciona como /p2p, mas automaticamente solicita transferencia de arquivo quando ambos entram no lobby. O peer precisa aceitar.

Examples

/sendfile Mario
